  • OMORI for playstation vita est un portage du jeu OMORI sur playstation vita utilisant le moteur OpenRPGENGINE.

    OpenRPGENGINE est un moteur de jeu développé par mes soins, basé sur lua player plus.

    Il permet l'implémentation de nouvelles fonctionalités permettant la simplification du développement de JRPG en lua sur psvita.

    De nouvelles versions du moteur permettant son fonctionnement sur d'autres plateformes sont en cours de développement.

    Ce portage implémentera,grâce au moteur OpenRPGENGINE, de nouvelles fonctionnalités telles que,

    l'affichage au format 16/9, l'amélioration de la maniabilité ou encore l'amélioration de la qualité graphique des sprites.

fichiers du projet
Le code et l'ensemble des fichiers de développement du projet est disponible ici
vous trouverez dans l'archive les assets du jeux ainsi que trois programmes lua:
  • events.lua : programme lua contenant l'ensemble des fonctions utilisées dans les instances d'objets de type event
  • openRPGEngine.lua : programme lua contenant le moteur OpenRPGENGINE.
    il est appelable par un autre programme lua via la commande dofile("emplacementMoteur/openRPGEngine.lua")
  • index.lua: le jeu en lui même
